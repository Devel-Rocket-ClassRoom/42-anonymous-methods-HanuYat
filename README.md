# 익명 메서드

## 사용 방법

1. Visual Studio에서 `Chapter42-AnonymousMethods.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice-01 | 교안 코드 따라치기 - 명명된 메서드, 익명 메서드 기본, Func/Action, 매개변수 생략 (6개 과제) |
| CodingPractice-02 | 교안 코드 따라치기 - 익명 메서드 인스턴스, 변수 캡처, 클로저 (5개 과제) |
| CodingPractice-03 | 교안 코드 따라치기 - 람다식 비교, 이벤트 초기화, 콜백 (4개 과제) |
| CodingPractice-04 | 교안 코드 따라치기 - 표현식 비교, 조건부 필터링, 정적 익명 메서드 (3개 과제) |
